# Namma Railu Buddy

Namma Railu Buddy is an Android application designed to help passengers navigate local trains بسهولة by providing platform information, coach positions, and real-time updates.

## Problem
Local train passengers often face:
- Unclear or last-minute platform information
- Difficulty hearing announcements
- Confusion about coach positions
- Missing trains due to lack of timely updates

## Solution
Namma Railu Buddy acts as a passenger guide by offering:
- Platform numbers
- Coach position details (General, Ladies, etc.)
- Crowdsourced delay and arrival updates
- Location-based destination alerts

## Features
- Live station selection
- Real-time platform updates (crowdsourced)
- Coach position guidance
- Wake-up alarm based on destination proximity
- Simple and user-friendly interface

## Tech Stack
- Android Studio (Java/Kotlin)
- Firebase for real-time data
- GPS / Geofencing for location tracking

## User Flow
1. Select your station
2. View train and platform information
3. Check coach position
4. Set destination alarm
5. Receive alert before arrival

## Impact
- Reduces confusion for daily commuters
- Saves time and improves travel efficiency
- Makes railway navigation easier for first-time users
- Enhances overall public transport experience
